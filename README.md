# gh-pages-sample

GitHub Pagesに自分のポートフォリオサイトを

## 詳細

`https://yoshikawa.github.io` にポートフォリオサイトを作ります

学部生の参考になるポートフォリオサイトになるくらいの簡単な感じで

## 動作環境

|言語/FW|Version|
|:--:|--:|
|node|14.5.0|
|npm|6.14.6|

## 使い方

```sh
# 開発する際のコマンド
$ npm run-script start

# ウェブサイトのビルドをするコマンド
$ npm run-script build
```
